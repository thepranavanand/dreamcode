1. 'p' tag : is used to add a paragraph.

2. 'strong' tag : is used to make any text bold.
Note : We can also use 'b' tag for bold but 'strong' is preferred.

3. 'H' tag : is used for adding Heading and subheadings to a page.
We can write them H1, H2, H3, etc.

4.'br' or '/br' or 'BR' or '/BR' tag : is used for jumping to the next line.

5.'em' tag also called emphasis tag : is used to make any text italic.

6. <a href = "link to the website">Title</a > is used for redirecting to a website.

7.Main tags are : html --> head --> title --> /title --> /head--> body --> /body-->/html

8. We can attach images either from the internet or a local image from the same folder where this html page is located by using : <img src = "link to the image or location of image in folder."/>

9. We can redirect to a link by clicking on an image through this method : <a href = "link to the page"> Title<img src = "link to the image"/></a>
